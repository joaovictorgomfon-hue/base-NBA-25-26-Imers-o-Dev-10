# Projeto Base NBA — Imersão Dev

Este projeto é um primeiro mergulho nos conceitos fundamentais de **HTML**, **CSS** e **JavaScript**, usados para construir e manipular interfaces.

- **HTML:** estrutura os elementos visíveis da aplicação  
- **CSS:** define estilo, layout e organização  
- **JavaScript:** adiciona lógica, interação e integração com APIs  

Também inclui o uso de um banco de dados de times da NBA, carregado a partir de um arquivo `data.json`, simulando o consumo de uma API pública e organizando os dados para uso no projeto.

## Objetivos do projeto

A ideia é entender, na prática, como:

- **Consumir dados externos** via fetch  
- **Processar e organizar** a resposta JSON  
- **Renderizar** os dados no front-end usando JavaScript  
- **Estruturar a aplicação** com HTML e CSS
